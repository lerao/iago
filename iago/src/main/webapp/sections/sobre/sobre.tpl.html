Sobre


O Portal de Dados Abertos da Universidade Federal de Pernambuco oferece dados públicos que podem ser livremente acessados e utilizados por toda a comunidade. Os dados disponíveis no portal são publicados seguindo boas práticas internacionais e englobam diferentes categorias: dados acadêmicos, administrativos e orçamentários.

O Portal de Dados Abertos é de responsabilidade da PROCIT, sendo mantido pelo Laboratório de Dados e Informações (aladin) e pelo Núcleo de Tecnologia da Informação (NTI) da UFPE. Para a publicação dos conjuntos de dados, é utilizado o Iago, um Sistema Gerenciador de Dados na Web (SGDW). O Iago permite um gerenciamento mais completo dos dados publicados, englobando todas as etapas do ciclo de vida dos dados na Web (abaixo).
